
# Super Babylon Bros - Level Design Document

## 1. Physics Constraints (The "Rules of Construction")
Before designing maps, we must adhere to the current physics engine settings:
*   **Gravity:** -60.0 (Very heavy/fast falling). **EXCEPTION:** World 1-4 is Underwater (Gravity -10.0).
*   **Move Speed:** 9.0
*   **Jump Force:** 26.0
*   **Max Jump Height:** ~4.5 Blocks vertical.
*   **Max Jump Distance:** ~5.5 Blocks horizontal (running).
*   **Safe Jump Distance:** 3-4 Blocks (Design target for fairness).

## 2. Legend
*   `#` : Ground / Wall (Indestructible)
*   `K` : Brick Block (Breakable, might contain items) or Koopa Enemy in map context
*   `?` : Question Block (Contains Coin or Flower)
*   `T` : Pipe (Decoration/Obstacle)
*   `E` : Goomba (Red Enemy - Stompable)
*   `S` : Spiky (Blue Enemy - Hurts on touch, needs Fireball)
*   `G` : Ghoul (Green Monster - Fast speed)
*   `F` : Fish (Red Swimming Enemy - Underwater only)
*   `K` : Koopa (Green Turtle - Patrolling)
*   `C` : Coin (Collectible)
*   `L` : Lava (Instant Death)
*   `F` : Flagpole (Level End)
*   `P` : Player Start Position

## 3. World 1-1: The Plains
**Theme:** Open air, green grass, blue sky.
**Goal:** Basic movement.

## 4. World 1-2: The Underground
**Theme:** Enclosed, lava, dangerous.
**Goal:** Precision jumping.

## 5. World 1-3: The Haunted Peaks
**Theme:** High altitude, floating islands.
**Enemy:** Ghoul.

## 6. World 1-4: The Deep Blue (NEW)
**Theme:** Underwater.
**Physics:** Low Gravity (-10), High Drag. Infinite Jumping (Swimming).
**Enemy:** Fish (F).
**Layout:** Open water with coral reefs (blocks) and suspended coins.
